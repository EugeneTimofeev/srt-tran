# srt-tran
Translate SRT files from EN to RU with Google Translate
Простой пример работы с Google Translate на Golang.
Программа принимает аргументами папки, в которых есть файлы субтитров SRT, переводит их, создает рядом файлы русских субтитров с именем, аналогичным исходному, но с суффиксом '_rus'.
Папки могут быть вложенными, глубина вложенности любая.
